## 📋 Sobre o Projeto

O **Soberania Verde** é uma página web informativa criada com o objetivo de conscientizar e educar produtores rurais e a comunidade sobre o conceito de **Fazenda Autossuficiente**. O projeto aborda como o agro de energia limpa permite fechar um ciclo 100% sustentável, transformando o que antes era considerado resíduo em força de trabalho e eletricidade.

O site destaca de forma interativa os três pilares principais dessa revolução verde:
1. **Biodigestores:** Transformação de dejetos em biogás e biofertilizantes.
2. **Energia Solar:** Uso de placas fotovoltaicas e o conceito inovador de Agri-Voltaica.
3. **Biomassa e Biocombustíveis:** Aproveitamento de matéria orgânica e óleos vegetais.

---

## 🚀 Funcionalidades

* **Layout Responsivo:** Adaptado para leitura em computadores, tablets e celulares.
* **Componente Accordion:** Organização limpa e interativa para explicar os pilares de energia limpa.
* **Galeria de Fotos com Lightbox:** Visualização ampliada das plantações e paisagens da região do Paraná, com opção de download das imagens.
* **Seção "Saiba Mais":** Curadoria de artigos científicos para quem deseja se aprofundar no tema.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais (Vanilla Stack):

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout responsivo e transições.
* **JavaScript (Vanilla):** Lógica do Accordion e funcionamento do Lightbox de imagens.
